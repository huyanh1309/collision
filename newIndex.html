INFO PAGE

1. Introduction

- This is a test conducted to calculate the digits of the Pi number

2. Purpose:

- The purpose of this is to calculate the digits of the Pi number using elastic collision between 2 objects.

3. Mechanics:

- The test was conducted using the elastic collision formula.

In which, the elements are:

- u1, u2: The velocities before collision.

- v1, v2: The velocities after collision.

- m1, m2: The mass of the objects.
